# 아서앤그레이스 고객관리시스템

아서앤그레이스 브랜드의 고객 정보를 효율적으로 관리할 수 있는 웹 기반 시스템입니다.

## 🌟 주요 기능

- **고객 정보 관리**: 고객 등록, 수정, 삭제
- **구매 이력 추적**: 상세한 구매 기록 관리
- **선물 이력 관리**: 고객별 선물 제공 내역
- **방문 주기 분석**: 고객 방문 패턴 추적
- **등급 시스템**: 총 구매액 기준 자동 등급 분류 (VVIP/VIP/일반)
- **생일 알림**: 이번 달/다음 달 생일 고객 알림
- **엑셀 데이터 처리**: 대량 데이터 업로드/다운로드
- **📱 모바일 최적화**: 반응형 디자인으로 모바일에서도 편리하게 사용
- **☁️ 실시간 클라우드 동기화**: 모든 기기에서 자동으로 데이터 동기화
- **🔄 테이블 정렬**: 고객 목록을 이름, 등급, 생년월일 등으로 정렬 가능

## 🚀 실시간 동기화 설정 가이드

### 1단계: JSONBin 계정 생성 및 설정

1. **JSONBin.io 가입**
   - [JSONBin.io](https://jsonbin.io)에 접속
   - 무료 계정 생성 (GitHub/Google 로그인 가능)

2. **새 Bin 생성**
   - 대시보드에서 "Create Bin" 클릭
   - 빈 JSON 객체 `{}` 입력
   - Bin 이름을 "고객관리데이터"로 설정
   - "Create" 클릭

3. **API 키 생성**
   - 좌측 메뉴에서 "API Keys" 클릭
   - "Create Access Key" 클릭
   - Key 이름을 "고객관리시스템"으로 설정
   - "Create" 클릭 후 생성된 키 복사

4. **Bin ID 확인**
   - 생성된 Bin을 클릭
   - URL에서 Bin ID 확인 (예: `65f1a2b3c9e77c0012345678`)

### 2단계: 시스템에서 동기화 활성화

1. **고객관리시스템 접속**
   - 웹사이트에 로그인

2. **동기화 설정 열기**
   - 상단 네비게이션에서 "🌤️ 동기화 설정" 클릭

3. **설정 정보 입력**
   - **API URL**: `https://api.jsonbin.io/v3/b/YOUR_BIN_ID`
     (YOUR_BIN_ID를 실제 Bin ID로 교체)
   - **API Key**: JSONBin에서 생성한 Master Key 입력
   - "동기화 활성화" 클릭

### 3단계: 동기화 확인

1. **테스트 데이터 입력**
   - PC에서 고객 정보 추가

2. **다른 기기에서 확인**
   - 모바일/태블릿에서 같은 웹사이트 접속
   - 10초 후 자동으로 데이터 동기화 확인

## 💡 동기화 작동 원리

### 📊 실시간 동기화 특징
- **자동 감지**: 다른 기기에서 데이터 변경 시 10초 내 자동 감지
- **충돌 방지**: 타임스탬프 기반으로 최신 데이터 우선 적용
- **기기 구분**: 각 기기에 고유 ID 부여로 무한 루프 방지
- **즉시 저장**: 데이터 변경 시 즉시 클라우드에 저장

### 🔄 동기화 상태 표시
- ✅ **동기화 완료**: 최신 데이터로 업데이트됨
- 🔄 **동기화 중**: 데이터를 업데이트하는 중
- ❌ **동기화 실패**: 네트워크 오류 또는 설정 문제
- 📶 **오프라인 모드**: 로컬 데이터만 사용

## 🔧 기술 스택

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **UI Framework**: Bootstrap 5.3
- **Icons**: Bootstrap Icons
- **PDF 생성**: jsPDF

- **클라우드 동기화**: JSONBin API
- **정렬 기능**: 한국어 localeCompare 지원

## 🎯 등급 시스템

- **VVIP**: 총 구매액 2,000만원 이상 (보라색 배지)
- **VIP**: 총 구매액 1,000만원 이상 (빨간색 배지)
- **일반**: 그 외 모든 고객 (파란색 배지)

등급은 구매 기록 추가/수정/삭제 시 자동으로 업데이트됩니다.

## 🌐 배포 방법

### GitHub Pages
```bash
1. GitHub 저장소에 코드 업로드
2. Settings → Pages → Source를 main branch로 설정
3. 생성된 URL로 접속
```

### Netlify
```bash
1. Netlify에 저장소 연결
2. netlify.toml 파일 자동 인식
3. 자동 배포 완료
```

### Vercel
```bash
1. Vercel에 저장소 연결
2. vercel.json 설정 파일 자동 인식
3. 자동 배포 완료
```
